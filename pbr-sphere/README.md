# DirectX 11 PBR Sphere Application

DirectX 11 application rendering a sphere-based scene with point-light PBR shading and a cubemap environment.

## Features

- Sphere grid with roughness and metalness variation across the scene
- Physically Based Rendering for three point lights
- Display modes for full PBR, normal distribution function, geometry function, and Fresnel function
- Keyboard switching between display modes with digit keys
- Simple cubemap-based environment background and reflections
- Clamp usage in shader code for material and lighting terms with bounded ranges
